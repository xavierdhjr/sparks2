Summary

This system manages the stages a crop goes through and the rewards of harvesting a crop.

Goals

  • Manage the stages of growth of a GridObject that is marked with the Capability CropHarvestable.
  • Give rewards (EXP, items) for harvesting crops.
  • Consume resources demanded by a harvest (Energy).

Data Format

How to create a CropHarvest in XML

<!-- Harvestable Item -->
<farmitem>
<name>Field of Wheat</name>
<item-id>farm_item_1a</item-id>
</farmitem>
<!-- Produced Item -->
<farmitem>
<name>Wheat</name>
<item-id>farm_item_1</item-id>
<art-asset>wheat</art-asset>
</farmitem>
<!-- Harvestable GridObject -->
<gridobject item-id="farm_item_1a" width="10" height="10">
<art-asset>GridObjects/farm_item_1a</art-asset>
<art-asset>GridObjects/farm_item_1b</art-asset>
<art-asset>GridObjects/farm_item_1c</art-asset>
<capability>draggable</capability>
<capability>highlightable</capability>
<capability>cropHarvestable</capability>
<layer>plots</layer>
</gridobject>
<!-- Harvest Info -->
<cropharvest item-id="farm_item_1a" energy="1" time="5" mexp="1">
<produces item-id="farm_item_1" qty="1" experience="2"/>
</cropharvest>

Requirements

  • An item that represents the GridObject
  • The produced item (What you get from Harvesting)
  • The GridObject players can click on to Harvest
    • Must be tagged with <capability>cropHarvestable</capability>
  • A Harvest Info section that hooks together the GridObject and the produced item and defines EXP rewards, Energy requirements, and Time requirements.

How To Use

Determining if a GridObject can be Harvested

A GridObject can be harvested if it is marked with the Capability CropHarvestable.

// This will return false if the GridObject does not have
// the right Capability or if it is not "Ready".
bool canBeHarvested = CropHarvestSystem.Get.CanBeHarvested(Obj);

Determining the state of a Crop Harvest

CropHarvestSystem.CropHarvestState state = CropHarvestSystem.Get.GetHarvestStateFor(Obj);
if (state.CurrentStage == CropHarvestSystem.Stage.Growing)
{
Debug.Log("Crop is growing. Time Left:" + state.GetTimeLeftString());
}
else if (state.CurrentStage == CropHarvestSystem.Stage.AwaitingMaintenence)
{
Debug.Log("Crop is seeding. Time Left:" + state.GetTimeLeftString());
}
else
{
Debug.Log("Crop is ready to be maintained / harvested.");
}

This example puts it all together and tries to harvest a referenced GridObject whenever the mouse is clicked.

Note that you need to Register a GridObject with the CropHarvestSystem before it can record any CropHarvestState information.

using UnityEngine;
using System.Collections;
public class CropHarvestExample : MonoBehaviour
{
public GridObject Obj;
// Use this for initialization
void Start ()
{
// This will return false if the GridObject does not have
// the right Capability or if it is not "Ready".
bool canBeHarvested = CropHarvestSystem.Get.CanBeHarvested(Obj);
// This object must be registered with the CropHarvestSystem
// otherwise no state information will be recorded.
if(canBeHarvested)
CropHarvestSystem.Get.RegisterHarvestableObject(Obj);
}
void Update()
{
if (Input.GetMouseButtonDown(0))
CropHarvestSystem.Get.Harvest(Obj);
CropHarvestSystem.CropHarvestState state = CropHarvestSystem.Get.GetHarvestStateFor(Obj);
if (state.CurrentStage == CropHarvestSystem.Stage.Growing)
{
Debug.Log("Crop is growing. Time Left:" + state.GetTimeLeftString());
}
else if (state.CurrentStage == CropHarvestSystem.Stage.AwaitingMaintenence)
{
Debug.Log("Crop is seeding. Time Left:" + state.GetTimeLeftString());
}
else
{
Debug.Log("Crop is ready to be maintained / harvested.");
}
}
}
