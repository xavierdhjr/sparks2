void ObjectPoolController< T >.ReleaseObject ( T  o)

Frees up an object in the pool. This deactivates the GameObject and removes it from the cache if this pool is caching its GameObjects.

Parameters
oObject to release.
NeverUseAgainIf true, this relased object will not be able to be requested again.

Member Data Documentation

bool ObjectPoolController< T >.OneTimeUseObjects = false

Setting this to true will cause objects to be released permanently instead of allowing them to be recycled.

This is useful when resetting all properties on an object is more difficult or costly than allocating the exact number needed.
